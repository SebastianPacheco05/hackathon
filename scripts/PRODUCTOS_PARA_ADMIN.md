# Script de Productos para Crear Manualmente desde Admin

## Resumen Ejecutivo

Este documento contiene **50 productos** listos para crear manualmente desde el panel de administración. Cada producto incluye:

- ✅ Información básica (nombre, slug, categoría, marca, proveedor)
- ✅ Descripción del producto
- ✅ **3 a 5 grupos de variantes** por producto
- ✅ Cada grupo tiene **1 a 3 combinaciones** con SKU, precio y stock
- ✅ Atributos adicionales cuando aplica (colores, tallas, almacenamiento, etc.)

### Estadísticas Generales

- **Total de productos:** 50
- **Categorías utilizadas:** 14
- **Marcas utilizadas:** 17
- **Proveedores utilizados:** 13
- **Variantes por producto:** Entre 3 y 5 grupos
- **Combinaciones totales:** ~150+ variantes individuales

### Cómo Usar Este Documento

1. **Navega al panel de administración** → Gestión de Productos
2. **Para cada producto:**
   - Crea el producto base con la información proporcionada
   - Asigna la categoría, marca y proveedor indicados
   - Agrega la descripción
3. **Para cada grupo de variantes:**
   - Crea un grupo de variante con el atributo dominante (ej: Color)
   - Asigna el valor dominante (ej: Verde, Blanco, etc.)
   - Si hay color hex, úsalo para el swatch visual
4. **Para cada combinación dentro del grupo:**
   - Crea la combinación con el SKU proporcionado
   - Asigna precio y stock
   - Agrega atributos adicionales si los hay (ej: Almacenamiento, Talla)

### Notas Importantes

- Los **SKUs** son únicos y están pre-generados
- Los **precios** están en pesos colombianos (COP)
- Los **stocks** son valores aleatorios entre 10-100 unidades
- Los **IDs de categoría, marca y proveedor** son los reales de tu base de datos
- Los productos están basados en **categorías reales** de tu catálogo
